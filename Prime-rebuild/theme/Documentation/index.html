Installation How to Install Route

1) Download & Organize Files

First of all thank you for purchasing our theme.Before going to the installation part, let us briefly discuss the internal structure of theme. After you download ZIP file from ThemeForest UNIZP that file, after unziping you see various structure of theme.
  • theme/route.zip (Root file to install wordpress theme)
  • theme/route/ (Directory use to install the wordpress theme via FTP)
  • Documentation (Directory which includes theme documentaion file)
  • Licensing (File which include information about wordpress & theme under GPL liscense)
  • Resources (Directory which includes fla file, xml dump, jquery file, etc)

2) Installing Theme

This section will introduce how to install the wordpress theme by various methods, Basically there are two methods of installing wordpress theme. Let's introduce two ways briefly
  • Installing your theme via Wordpress
  • Installing your theme via FTP

Installing your theme via Wordpress

  1. Login to your wordpress dashboard, go to WordPress Appearance -> Themes section.
  2. Click on Install Themes tab.
  3. Click on Choose File button, and select .zip theme file and click Install Now button.
  4. If you uploading process fails due to theme broken issues, then you may refer here
Note : This method will only work if your server allows space greater than 5 MB. Usually hosting providers provides only 2 MB

Installing your theme via FTP

  1. Download the Theme archive .zip from ThemeForest and extract route.zip the files it contains.
  2. Move extracted files to /wp-content/themes/route via FTP Clients
  3. For example theme should be in wp-content/themes/route
  4. Follow the instruction below for activating the theme.

3) Activating Theme

To select a Theme for your site
  1. Log in to the WordPress Administration Panels.
  2. Select the Appearance panel, then Themes.
  3. From the Available Themes section, click on the route title (or route screenshot) to activate.
  4. A preview of the Theme will be shown.
  5. To activate the Theme click the Activate route link in the top right.

4) Things to Remember After Installation

After complete the installation, there is a crucial part you must finish. This part mainly operates your web site.
Note : /wp-content/themes/route/cache/ with CHMOD777 Permission (if it does not work set CHMOD 755)

This is setting CACHE folder permission, also maintains many parts of the web site to work.For more information check file permission mode.

5) Import Demo Content

We are providing you a XML file, which consists of all demo content included in live preview, so to activate this file follow these steps.
  1. Find the resource folder inside the root folder of route Theme.
  2. Inside resource folder there are 3< files, all-demo.xml pages-demo.xml posts-demo.xml
  3. If you want to include only pages then selecct pages-demo.xml or only posts then posts-demo.xml or select all-demo.xml for all demo content.

6) Theme Updation

If you want further notice about updation of theme, then please follows us on twitter or constantly visit our support forum.
